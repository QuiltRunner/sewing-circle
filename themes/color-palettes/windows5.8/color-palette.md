# Windows5.8 Color Palette

This palette reflects the retro-terminal, EAS broadcast, neon-alert aesthetic of the **Windows5.8** theme.  

##  Windows5.8 Color Palette

- `#05070B - Black Blue (Almost black with a faint hint of navy)`
- `#D6D7E0 - Light Gray (Soft, cool gray with blue undertones)`
- `#4EF2FF - Bright Cyan (Vivid, electric blue-green)`
- `#5CFF8D - Neon Green (Bright green with high visibility)`
- `#FF3D6E - Hot Pink (Intense pink-red)`
- `#FFE86C - Soft Yellow (Bright pastel yellow with warm underglow)`
- `#1B4C9E - Royal Blue (deep medium blue with strong saturation)`
- `#7B58C6 - Purple Violet (Medium purple with cool undertones)`

## Notes
- Colors were chosen to match the key UI elements: CRT blacks, neon cyan, emergency red-magenta, console gray, terminal blues, etc.
- These values work well for theming terminals, prompts, and UI elements
