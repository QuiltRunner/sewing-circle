# Windows 5.8 EAS / Government Alert Wallpaper

This wallpaper is a satrical take on Emergency Alert System aesthetics mixed with 80s terminal screens, ARG-style government notices, and some inside jokes about our dogs.

## Visual Style

- EAS broadcast interruption vibes
- Retro CRT terminal displays and monochrome text
- Neon typography and "glitch" accents
- Bell Atlantic payphone UI mockup
- "Google '84" retro-propaganda terminal block
- FBI-style dossiers featuring our 2 dogs and their antics
- Color palette: terminal greens, blues, and pinks with red accents

## Creation & Tools

- Designed in **Canva Premium**
- Layered composition: CRT textures, glitch overlays, static noise, terminal windows, warning banners, and mock government notices
- Canva-licensed graphic elements and assets

## Format & Resolution

- 4k UHD wallpaper
- Optimized for dark-theme desktops
- Text is readable at a distance, even with transparent windows layered on top
