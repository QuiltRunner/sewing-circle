# Vaporwave F1 Desktop Wallpaper

This wallpaper blends a full vaporwave spectrum (lavenders, pinks, cyans, and neon gradients) with **F1 analytics**, **vintage PCs**, **maximalism**, and **retro-tech energy**.
It's designed to feel like an F1 engineering desk in a pastel dreamland.

## Visual Style

- Pastel vaporwave with soft neon overlays
- Retro desktop technology (CRT monitors, floppy disks, old-school pop-ups, and graph displays)
- F1 track map (Mexico City), DRS zones, race telemetry-style charts
- Maximalism 
- Legal iconography (Lady Justice)

## Creation & Tools

- Built in **Canva Premium**
- Composite of multiple layers: gradients, illustrations, custom color overlays, and thematic icons
- Various assets sourced from Canva's licensed library
- General inspiration drawn from vaporwave media, motorsport graphics, and retro desk setups

## Format & Resolution

- 4k wallpaper
- Optimized for ultrawide and dual-monitor setups
- High-contrast details preserved under transparent or blurred terminal windows
