# Themes

This directory contains all visual assets that shape the overall aesthetic of my systems. 

Most of these are custom or semi-custom creations aligned with my preferred vibes:
- Pastel
- Vaporwave
- 80s EAS horror
- F1-inspiration
- Tarot and occult symbolism
- 90s girly-pop

## Structure

### color-palettes/

Collections of color sets used for terminals, Plasma, Kvantum, and themed utilities. 

### wallpapers/

Wallpapers curated or created for consistent visual identity across each system. 

### assets/

Logos, icons, overlays, SVGs, and miscellaneous theme components. 

**Everything here is fair game for remixing. If you adapt / change something, I would love to see what you made!**
