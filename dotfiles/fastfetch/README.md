# Fastfetch Configurations

System info displays, logos, and custom module configurations for Fastfetch.

## Files

- `config.jsonc`
- Custom ASCII / Unicode logos
- Experimental configs

## Installation

Copy or symlink `config.jsonc` to `~/.config/fastfetch`.

## Notes 

Fastfetch versions can differ based on the release you're using. Release notes will be added for configuration files.
