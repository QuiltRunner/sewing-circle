# Kvantum Theme Configurations

Kvantum themes (.kvconfig and .svg) used across KDE Plasma setups.

## Files
- Theme SVGs
- `.kvconfig` configuration files
- Custom color tweaks

## Installation

1. Copy themes folder into `~/Kvantum/themes` or `~/.local/share/Kvantum/themes`.
2. Run Kvantum Manager by using `kvantummanager` in your terminal or via your DE's application menu.
3. Select the theme folder and adjust to your liking.

## Notes

Themes here often match Plasma, Kitty, and Fastfetch colors for aesthetic cohesion.
